## Hobby Project Planner

This is where the code of the app will be stored. 

It has the app's pages:
- Home
- Projects
-  Profile

From here, further development can start.

Before running main.py, run 'pip install -r requirements.txt' this will download all necessary libraries. Make sure you are in the correct path. Once downloaded, just do 'python main.py' and it will run.
